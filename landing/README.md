# Run and deploy the landing page

This contains everything you need to run your app locally.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Local URL
- `http://localhost:3001`

## Routes
- `/` Home landing page
- `/updates` release notes page
